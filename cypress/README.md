# Cypress E2E Testing – UlovDomov.cz

This project demonstrates professional End-to-End test automation using Cypress, designed for a real estate web platform.  
Focus is on form validation, user flows, and backend request monitoring in a fully interactive multi-step listing process.

---

## What’s being tested

✔ User authentication flow  
✔ Full offer creation: from landing page → listing type → property details → publishing  
✔ Client-side form validations and UI controls  
✔ API request interception, assertions, response validation  
✔ File upload handling  
✔ Navigation and URL verification

---

## Tech Stack

- **Cypress 13** – E2E automation framework
- **JavaScript (ES6)**
- **Mocha & Chai** – assertions & test structure
- **Page Object Model (POM)** – reusable selectors
- **Custom utilities** – request tracking & URL validation
